# QA bug reporting features

## High value

1. [ ] Screenshot - Capture the current tab
2. [x] Date and time - When the bug was captured
3. [ ] Page title - Document title
4. [ ] Device pixel ratio - For retina/scaling issues
5. [x] Display resolution - Full screen size (not just viewport)

## Medium value

6. [ ] Cookies enabled - Privacy/functionality issues
7. [ ] Language - Browser locale
8. [ ] Timezone - For time-related bugs
9. [ ] Network status - Online/offline
10. [ ] Memory usage - Performance issues

## Advanced

11. [ ] Console errors - JavaScript errors on the page
12. [ ] Local/Session storage size - Storage issues
13. [ ] Copy to clipboard - One-click copy all info
14. [ ] Export as JSON/Markdown - For bug tickets
