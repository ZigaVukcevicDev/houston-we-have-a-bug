# QA bug reporting features

## High value

- [x] Screenshot - Capture the current tab
- [ ] Crop screenshot
- [x] Date and time - When the bug was captured
- [x] Device pixel ratio - For retina/scaling issues
- [x] Display resolution - Full screen size (not just viewport)
- [x] Copy system info to clipboard

## Medium value

- [ ] Localization
- [ ] Language - Browser locale
- [ ] Memory usage - Performance issues

## Advanced

- [ ] Console errors - JavaScript errors on the page
- [ ] Export as JSON/Markdown - For bug tickets
- [ ] Submit bug to Azure DevOps
