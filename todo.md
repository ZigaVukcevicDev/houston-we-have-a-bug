# QA bug reporting features

## High value

1. [ ] Screenshot - Capture the current tab
2. [x] Date and time - When the bug was captured
3. [x] Device pixel ratio - For retina/scaling issues
4. [x] Display resolution - Full screen size (not just viewport)

## Medium value

5. [ ] Cookies enabled - Privacy/functionality issues
6. [ ] Language - Browser locale
7. [ ] Timezone - For time-related bugs
8. [ ] Network status - Online/offline
9. [ ] Memory usage - Performance issues

## Advanced

10. [ ] Console errors - JavaScript errors on the page
11. [ ] Local/Session storage size - Storage issues
12. [ ] Copy to clipboard - One-click copy all info
13. [ ] Export as JSON/Markdown - For bug tickets
